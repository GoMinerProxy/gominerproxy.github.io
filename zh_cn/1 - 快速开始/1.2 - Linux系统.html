1.2 - Linux系统

Linux下 GoMinerProxy 的搭建 *快速一键脚本可以参考[3.4章]

兼容性

实测兼容绝大部分64位的Linux系统，如Centos7/8、Ubuntu16/18/20/21、Debian9/10、Archlinux等系统。

获取 GoMinerProxy

你可以在 GitHub Release 页面获取已经构建打包完成的主程序。其中每个版本都提供了常见系统架构下可用的主程序，命名规则为 GoMinerProxy_版本号_操作系统_CPU架构.tar.gz。比如，普通64位Linux系统上部署1.0.0版本，则应该下载GoMainerProxy_v1.0.0_linux_amd64.tar.gz。

启动 GoMinerProxy，并使用screen控制

1、先使用SSH连接上您的Linux服务器，然后安装 wget 下载工具和 screen 视窗管理程序(非精简系统会自带)：

# Centos 系统使用
yum install wget screen -y

# Debian/Ubuntu 系统使用
apt install wget screen -y

2、下载对应版本的 tar.gz 压缩包，或是手动上传也可以（最新版本请见 Github Release）：

# 下述下载地址为示例，请勿原样复制，请见 Github Release 右键复制最新版本和对应版本的链接
wget https://github.com/GoMinerProxy/GoMinerProxy/releases/download/1.0.0/GoMainerProxy_v1.0.0_linux_amd64.tar.gz

3、解压下载的 tar.gz 压缩包：

tar -zxvf GoMainerProxy_v1.0.0_linux_amd64.tar.gz

4、进入到子目录：

cd GoMainerProxy_v1.0.0_linux_amd64

5、赋予执行权限：

chmod 777 GoMinerProxy

6、使用 screen 防止SSH连接断开导致开启的 GoMinerProxy 一并关闭：

screen -S gominerproxy

7、启动 GoMinerProxy：

./GoMinerProxy

如果遇到报错的情况，请将显示的信息截图提交给 Telegram 讨论群组 内的开发者

Screen窗口常用指令

# 创建新窗口
screen -S <窗口名>

# 进入指定窗口
screen -r <窗口名>

# 最小化当前进入的窗口
在当前界面按下ctrl+a+d

# 关闭指定窗口
screen -X -S <窗口名> quit
