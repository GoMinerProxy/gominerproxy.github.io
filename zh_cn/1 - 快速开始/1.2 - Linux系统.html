1.2 - Linux系统

Linux下 GoMinerProxy 的搭建

兼容性

实测兼容绝大部分64位的Linux系统，如Centos7/8、Ubuntu16/18/20/21、Debian9/10、Archlinux等系统。

获取 GoMinerProxy

你可以在 GitHub Release 页面获取已经构建打包完成的主程序。其中每个版本都提供了常见系统架构下可用的主程序，命名规则为 GoMinerProxy_版本号_操作系统_CPU架构.tar.gz。比如，普通64位Linux系统上部署1.0.5版本，则应该下载GoMinerProxy_v1.0.5_linux_amd64.tar.gz。

Linux 一键脚本（没有Linux经验建议使用Windows系统）

# 先安装curl/wget/screen软件
# Centos执行 yum install curl wget screen -y
# Ubuntu执行 apt-get update -y && apt install curl wget screen -y

# 下述脚本均需要在使用安装脚本时相同的路径执行
# 一键安装GoMinerProxy
bash <(curl -s -L https://raw.githubusercontent.com/GoMinerProxy/GoMinerProxy/main/scripts/install.sh)
# 一键卸载GoMinerProxy
bash <(curl -s -L https://raw.githubusercontent.com/GoMinerProxy/GoMinerProxy/main/scripts/uninstall.sh)
# 一键更新GoMinerProxy到最新版本，可以不关闭GoMinerProxy闪断更新
bash <(curl -s -L https://raw.githubusercontent.com/GoMinerProxy/GoMinerProxy/main/scripts/update.sh)
# 服务器重启后打开 GoMinerProxy
bash <(curl -s -L https://raw.githubusercontent.com/GoMinerProxy/GoMinerProxy/main/scripts/start.sh)
# 重启 GoMinerProxy
bash <(curl -s -L https://raw.githubusercontent.com/GoMinerProxy/GoMinerProxy/main/scripts/restart.sh)
```

# 再次SSH链接可以使用下述指令查看输出
screen -r go_miner_proxy

Linux 详细部署过程（仅供参考）

# 下述下载地址为示例，请勿原样复制，请见 Github Release 右键复制最新版本和对应版本的链接
wget https://github.com/GoMinerProxy/GoMinerProxy/releases/download/1.0.5/GoMinerProxy_v1.0.5_linux_amd64.tar.gz
# 解压 tar.gz 压缩包
tar -zxvf GoMinerProxy_v1.0.5_linux_amd64.tar.gz
# 赋予执行权限
chmod 777 GoMinerProxy
# 创建窗口
screen -S go_miner_proxy
# 启动 GoMinerProxy
./GoMinerProxy

# 再次SSH链接可以使用下述指令查看输出
screen -r go_miner_proxy

如果遇到报错的情况，请将显示的信息截图提交给 Telegram 讨论群组 内的开发者

Screen窗口常用指令

# 创建新窗口
screen -S <窗口名>

# 进入指定窗口
screen -r <窗口名>

# 最小化当前进入的窗口
在当前界面按下ctrl+a+d

# 关闭指定窗口
screen -X -S <窗口名> quit
