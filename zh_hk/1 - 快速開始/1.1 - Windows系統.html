1.1 - Windows系統

1.1 - Windows系統

Windows下 GoMinerProxy 的搭建

兼容性

實測兼容絕大部分64位的Windows系統，如Windows Server 2008/2012/2016/2019/2022、Windows 7/8/8.1/10/11等系統。

獲取 GoMinerProxy

您可以在 GitHub Release 頁面獲取已經構建打包完成的主程序。其中每個版本都提供了常見系統架構下可用的主程序，命名規則為GoMinerProxy_版本號_操作系統_CPU架構.zip。比如，普通64位Windows系統上部署1.1.0版本，則應該下載GoMinerProxy_v1.1.0_windows_amd64.zip。

啟動 GoMinerProxy

使用 Winrar 或 7z 等解壓工具，直接解壓下載的 zip 壓縮包。

然后雙擊運行啟動 GoMinerProxy.exe ，或者在cmd內輸入 GoMinerProxy.exe 的文件路徑運行均可。

如圖提示后即可在瀏覽器內打開：http://<ip or domainname>:9998，即可打開后臺管理網頁。

注：如果網頁打開不了，請檢查云服務器處的防火墻是否開放對應端口，或聯系服務商。

如果遇到閃退的情況，請右鍵開始菜單-運行，輸入cmd運行，將主程序的路徑輸入或拖入到命令框中運行，再將顯示的信息截圖提交給 Telegram 討論群組 內的開發者
